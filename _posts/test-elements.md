Это тестовая страница, содержащая все поддерживаемые элементы Markdown. Она нужна для проверки корректности отображения стилей и работы парсера `markdown-it-py`.

## 1. Заголовки (Headings)

# Заголовок H1
## Заголовок H2
### Заголовок H3
#### Заголовок H4
##### Заголовок H5
###### Заголовок H6

---

## 2. Форматирование текста (Emphasis)

*   **Жирный текст** (Bold)
*   *Курсив* (Italic)
*   **Логическое *совмещение*** (Bold + Italic)
*   ~~Зачеркнутый текст~~ (Strikethrough)
*   `Инлайн-код` (Inline Code)

---

## 3. Списки (Lists)

### Маркированный список (Unordered)
*   Уровень 1
*   Уровень 1 (второй пункт)
    *   Уровень 2 (вложенность через 4 пробела)
    *   Уровень 2 (еще один пункт)
        *   Уровень 3
*   Возврат к Уровню 1

### Нумерованный список (Ordered)
1.  Первый пункт
2.  Второй пункт
    1.  Вложенный пункт 1
    2.  Вложенный пункт 2
3.  Третий пункт

---

## 6. Код (Code Blocks)

### Стандартный код (Светлая тема)
```python
def check_engine():
    status = "Luna SSG is running"
    for i in range(3):
        print(f"{status}... {i}")
    return True
```

### Терминал (Темная тема)
```bash
$ git status
On branch main
Your branch is up to date with 'origin/main'.

$ python main.py
Build complete! Created .nojekyll
```
